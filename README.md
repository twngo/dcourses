# dcourses
digital literacy course for NGOs and their volunteers / staff

數位能力培訓課程
==============

### 特色：切磋數碼技術與探討背後的社會問題、追求個體與社群平衡和共同利益

#### 適合對象：
- 想運用網路做點事情
- 喜歡思考，也能動手;不只拿也願意給的人


#### 期待成果：
- 寸有所長，知能交流，從彼此身上學到新知與技能
- 利用已有或新學到的技能為NGO/NPO for social goods
- 持續型的人力知識社群經營

專案網站：https://courses.twngo.xyz
